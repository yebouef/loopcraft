Loop Pattern — What should I pick?
Read both options, then click the one that fits.
Think of it like this: an AI agent is like a contractor you hire. The loop pattern decides whether you hire one person who does everything, or a project manager who runs a team of specialists.
🔄
Core Loop
One agent handles everything.
Use when:
  • One clear goal, one deliverable
  • Steps happen one after another
  • You want to control cost
📌 e.g. Write a SQL report, build a login page
🚀
Fleet Loop
An Orchestrator manages multiple Specialists.
Use when:
  • Multiple workstreams at once
  • Different expertise needed
  • Large or complex project
📌 e.g. Full platform: UI + API + DB + tests
🤔 Not sure? Can you describe your whole project in one sentence?

Loop Pattern Guide
🔄 Core Loop
  • One goal, one agent, one session
  • Steps run in sequence
  • Good for: reports, pages, scripts, emails
🚀 Fleet Loop
  • Orchestrator + multiple specialists in parallel
  • Different expertise at the same time
  • Good for: full platforms, multi-module systems
✅ End of Session Checklist
  • Save all outputs / files
  • Update Memory Snapshot (Step 3)
  • Commit & push to Git if it's a code project
  • Note what was decided + what's next
